# Uonetplus API

Unofficial Vulcan Uczniowie Optivium Net Plus API written with [FastAPI](https://fastapi.tiangolo.com/).
[Endpoint wiki](https://github.com/Marioneq4958/uonetplus_api/wiki/Logowanie)

## Installation
```sh
pip install -r requirements.txt
```

## Start the server
```sh
cd app
python -m main.py
```
